# 템플릿 캡처 GUI 사용법

`template_capture_gui.exe`는 AOS 단말 화면을 보면서 자동화에 사용할 버튼/팝업 영역을 PNG 템플릿으로 저장하는 도구입니다.

## 기본 흐름

1. `디바이스 확인`을 누릅니다.
2. `앱 목록`을 눌러 단말에 설치된 사용자 앱 목록을 가져옵니다.
3. 앱 리스트에서 실행할 앱을 클릭합니다.
4. `선택 저장` 또는 `앱 저장`을 눌러 실행 앱으로 저장합니다.
5. 필요하면 `앱 실행`을 눌러 선택 앱을 실행합니다.
6. `미러링 시작`을 눌러 실시간 단말 화면을 엽니다.
7. 자동화할 버튼이나 팝업이 보이면 `화면 캡처`를 누릅니다.
8. 캡처된 화면에서 저장할 영역을 마우스로 드래그합니다.
9. `템플릿 이름`과 `스크립트 이름`을 입력합니다.
10. `액션 <`, `액션 >`으로 저장할 액션을 선택합니다.
11. 액션에 값이 필요하면 `값` 입력칸을 클릭해서 입력합니다.
12. `scenarios.json 자동 등록`을 켠 상태에서 `템플릿 저장`을 누릅니다.
13. `스크립트 실행`을 누르면 선택된 스크립트만 실행됩니다.

## 액션 선택

- `탭`: 매칭된 템플릿 중심 좌표를 탭합니다.
- `탭 + 딜레이`: 탭 후 지정한 초만큼 대기합니다. `값`에는 초를 입력합니다.
- `탭 + 텍스트 입력`: 탭 후 텍스트를 입력합니다. `값`에는 입력할 텍스트를 입력합니다.
- `탭 + 아래 스크롤`: 탭 후 아래 방향 스크롤을 수행합니다. `값`에는 스와이프 거리(px)를 입력합니다.
- `탭 + 위 스크롤`: 탭 후 위 방향 스크롤을 수행합니다. `값`에는 스와이프 거리(px)를 입력합니다.

## 저장 예시

```json
{
  "name": "nickname_field",
  "script": "login_flow",
  "template": "templates/nickname_field.png",
  "threshold": 0.85,
  "action": {
    "type": "tap_text",
    "text": "qa_user",
    "delay_before_text": 0.3
  }
}
```

## 단축키

- `C` 또는 `R`: 화면 캡처
- `S`: 템플릿 저장
- `X`: 선택된 스크립트 실행
- `[` / `]`: 이전/다음 스크립트 선택
- `-` / `=`: 이전/다음 액션 선택
- `,` / `.`: 이전/다음 앱 선택
- `L`: 앱 목록 새로고침
- `U`: 현재 실행 중인 앱 선택
- `O`: scenarios.json 파일 열기
- `Q` 또는 `Esc`: 종료

## 참고

- 앱 리스트는 오른쪽 패널에서 클릭 선택할 수 있습니다.
- 미리보기 화면은 창 크기에 맞게 축소될 수 있지만, 저장할 때는 원본 캡처 좌표로 변환해서 자릅니다.
- 파일명에 사용할 수 없는 문자는 `_`로 바뀝니다.
- 같은 이름의 템플릿 파일이 있으면 `{이름}_1.png`, `{이름}_2.png`처럼 번호가 붙습니다.
